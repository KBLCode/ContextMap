---
description: "Show token usage statistics - 24h, 7d, 30d summaries with visual charts"
allowed-tools: ["Bash"]
---

Run the ContextViewer statistics command to display token usage:

```bash
~/.config/context-viewer/cmap.sh
```

Display the output directly to the user. This shows:
- 24h, 7d, 30d, all-time token summaries
- Hourly activity chart (24h)  
- 7-day trend chart
- Model breakdown with usage bars
- Recent sessions with usage bars
